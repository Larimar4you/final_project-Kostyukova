# Моделювання кидання двох гральних кубиків методом Монте-Карло

## Опис завдання

У цій роботі програмно реалізовано алгоритм моделювання кидання двох
гральних кубиків із використанням методу Монте-Карло.

Метою є:

- імітація великої кількості кидків двох кубиків;
- обчислення сум чисел, що випадають;
- визначення ймовірностей кожної можливої суми (від 2 до 12);
- аналіз збіжності експериментальних результатів
  до аналітичних ймовірностей при збільшенні кількості симуляцій.

---

## Метод Монте-Карло

Метод Монте-Карло полягає у багаторазовому випадковому моделюванні процесу
та подальшому статистичному аналізі отриманих результатів.

У даній задачі:

- кожен кидок кубиків моделюється за допомогою генератора випадкових чисел,
- для кожного кидка обчислюється сума чисел на кубиках,
- після виконання великої кількості симуляцій підраховується кількість
  появ кожної можливої суми,
- ймовірність кожної суми визначається як відношення кількості її появ
  до загальної кількості кидків.

## Реалізація

Для реалізації використано мову програмування **Python** та бібліотеки:

- `random` — для генерації випадкових чисел;
- `matplotlib` — для візуалізації експериментальних ймовірностей.

Програма виконує симуляції для різної кількості кидків:

- 100
- 1000
- 10 000
- 100 000
  Це дозволяє наочно продемонструвати, як зі збільшенням кількості
  експериментів результати методу Монте-Карло наближаються
  до теоретичних значень.

## Результати

У результаті виконання програми було:

- отримано експериментальні ймовірності кожної суми для різної кількості кидків,
- виведено числові значення ймовірностей у консоль,
- побудовано графіки розподілу ймовірностей для кожної серії симуляцій.

результати можуть суттєво відрізнятися від аналітичних значень,
однак зі збільшенням кількості симуляцій експериментальні
ймовірності стабілізуються та наближаються до теоретичних.
