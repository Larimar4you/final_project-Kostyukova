# Моделювання кидання двох гральних кубиків методом Монте-Карло

## Опис завдання

У цій роботі програмно реалізовано алгоритм моделювання кидання двох
гральних кубиків із використанням методу Монте-Карло.

Метою є:

- імітація великої кількості кидків двох кубиків;
- обчислення сум чисел, що випадають;
- визначення ймовірностей кожної можливої суми (від 2 до 12);
- порівняння отриманих результатів із аналітичними розрахунками.

---

## Метод Монте-Карло

Метод Монте-Карло полягає у багаторазовому випадковому моделюванні процесу
та подальшому статистичному аналізі отриманих результатів.

У даній задачі:

- кожен кидок кубиків моделюється за допомогою генератора випадкових чисел;
- після великої кількості симуляцій визначається частота появи кожної суми;
- ймовірність обчислюється як відношення кількості появ суми до загальної
  кількості кидків.

---

## Реалізація

Для реалізації використано мову програмування **Python** та бібліотеки:

- `random` — для генерації випадкових чисел;
- `collections.Counter` — для підрахунку частот;
- `matplotlib` — для візуалізації результатів.

Кількість симуляцій: **100 000 кидків**.

---
